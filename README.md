# PrecisionDx – AI-Based Brain Tumor Detection System

PrecisionDx is a deep learning–powered decision-support system designed to detect and classify brain tumors from MRI images.

## Features

- Two-stage CNN pipeline using Transfer Learning (ResNet50)
- Binary tumor detection + multi-class classification
- Grad-CAM explainability visualization
- Secure doctor authentication system
- Patient workflow management
- Automated PDF medical report generation

## Tech Stack

- Python
- Flask
- TensorFlow / Keras
- ResNet50
- HTML, CSS, JavaScript
- SQLite

## Model Details

- Dataset: 3000+ MRI images
- Accuracy: 94%
- Evaluation Metrics: Accuracy, Precision, Recall, F1-Score
- Explainable AI: Grad-CAM

## How to Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Open browser:
```
http://127.0.0.1:5000
```
